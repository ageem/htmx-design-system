HTMX Usage

CDN Include

<script src="https://unpkg.com/htmx.org@1.9.12"></script>

HTMX enhances HTML with attributes like hx-get, hx-post, hx-target, hx-swap.

Example Patterns

  1. Partial load: <button hx-get="/partials/snippet.html" hx-target="#out" hx-swap="innerHTML">Load</button>
  2. Form post: <form hx-post="/submit" hx-target="#out" hx-swap="innerHTML">...</form>
  3. Loading state: add hx-indicator="#spinner" and show a spinner while the request runs.
  4. Error state: handle htmx:responseError to show a toast or inline alert.
