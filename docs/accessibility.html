Accessibility

Keyboard

  • All interactive elements are reachable via Tab order.
  • Use :focus-visible outlines for clear focus states.
  • Dialog and menus should trap focus while open.

Roles & ARIA

  • Tabs: role="tablist", role="tab", and role="tabpanel" with aria-selected.
  • Icon-only buttons require an aria-label.
  • Inputs use associated <label for> and aria-describedby for help/error.

Reduced Motion

Motion durations collapse to 0 when the user prefers reduced motion, via @media (prefers-reduced-motion: reduce) tokens.
